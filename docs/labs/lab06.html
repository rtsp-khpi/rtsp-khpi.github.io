6. Програмування мовою Сі для ОС Linux¶

Призначення¶

Розробка програми «Гра в кості» мовою Сі для ОС Linux.

Опис¶

  1. Для автоматизації побудови програми використовувати утиліту make. Реалізувати цілі:

    • help,

    • release,

    • debug,

    • test,

    • dist,

    • install,

    • uninstall,

    • clean,

    • distclean.

  2. Дотримуватися Linux kernel coding style.

  3. Використовувати наступні назви директорій:

    • src - для розташування початкових кодів;

    • bin - для розташування результатів компіляції;

    • lib - для розташування створеної бібліотеки;

    • /opt/your_app_name - директорія для встановлення.

  4. Створити і використовувати статичну бібліотеку.

Увага

Приклад створення бібліотеки util

gcc -c util.c

ar -r libutil.a util.o

gcc main.c -o main -static -L./ -lutil

  1. Для зберігання результатів (ім’я користувача, результати підрахунку балів гравців, результат гри, відмітка про час) використовувати двоспрямовані списки.

  2. Режими гри визначати параметрами командного рядка.

  3. Розробити скрипт побудови і тестування (перевірка результатів виконання) програми.
