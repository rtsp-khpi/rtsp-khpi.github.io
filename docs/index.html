Увага

Set the Proxy for APT on Ubuntu

1) Create the following file:

/etc/apt/apt.conf.d/proxy.conf

2) Add the following lines:

Acquire::http::proxy «http://172.17.10.2:3128»;

Acquire::https::proxy «http://172.17.10.2:3128»;
